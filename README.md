# vibe-flow

## Setup

After cloning the repo, run:

```bash
uv sync
```

```bash
git config core.hooksPath .githooks
```

## Running

```bash
uv run vibe-flow
```

Alternative methods:

```bash
# Traditional: install in editable mode, then run
pip install -e .
vibe-flow

# Debug/development: run as module
python -m vibe_flow
```
