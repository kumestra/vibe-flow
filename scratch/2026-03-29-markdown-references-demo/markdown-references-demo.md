# Markdown References Demo

## 1. Link References

Define a link once, reuse it multiple times.

Visit [Google][google] or read the [GitHub Docs][gh-docs].
Later in the text, link to [Google][google] again without repeating the URL.

[google]: https://google.com
[gh-docs]: https://docs.github.com

---

## 2. Footnotes

Footnotes are useful for citations or side notes without cluttering the main text.[^1]

You can have multiple footnotes.[^2] They are rendered at the bottom of the document.

[^1]: This is the first footnote. It can contain detailed explanation.
[^2]: This is the second footnote.
